tooltip ​

基础用法 ​

文字气泡支持 12 个不同的方位。分别为：上左、上、上右、下左、下、下右、左上、左、左下、右上、右、右下。content属性设置提示文字，placement属性控制气泡弹出方向。

<template>
    <div class="tooltip-base-box">
        <div class="row center">
            <g-tooltip class="box-item" content="Top Left prompts info" placement="top-start">
                <g-button>top-start</g-button>
            </g-tooltip>
            <g-tooltip class="box-item" content="Top Center prompts info" placement="top">
                <g-button>top</g-button>
            </g-tooltip>
            <g-tooltip class="box-item" content="Top Right prompts info" placement="top-end">
                <g-button>top-end</g-button>
            </g-tooltip>
        </div>
        <div class="row">
            <g-tooltip class="box-item" content="Left Top prompts info" placement="left-start">
                <g-button>left-start</g-button>
            </g-tooltip>
            <g-tooltip class="box-item" content="Right Top prompts info" placement="right-start">
                <g-button>right-start</g-button>
            </g-tooltip>
        </div>
        <div class="row">
            <g-tooltip class="box-item" content="Left Center prompts info" placement="left">
                <g-button>left</g-button>
            </g-tooltip>
            <g-tooltip class="box-item" content="Right Center prompts info" placement="right">
                <g-button>right</g-button>
            </g-tooltip>
        </div>
        <div class="row">
            <g-tooltip class="box-item" content="Left Bottom prompts info" placement="left-end">
                <g-button>left-end</g-button>
            </g-tooltip>
            <g-tooltip class="box-item" content="Right Bottom prompts info" placement="right-end">
                <g-button>right-end</g-button>
            </g-tooltip>
        </div>
        <div class="row center">
            <g-tooltip class="box-item" content="Bottom Left prompts info" placement="bottom-start">
                <g-button>bottom-start</g-button>
            </g-tooltip>
            <g-tooltip class="box-item" content="Bottom Center prompts info" placement="bottom">
                <g-button>bottom</g-button>
            </g-tooltip>
            <g-tooltip class="box-item" content="Bottom Right prompts info" placement="bottom-end">
                <g-button>bottom-end</g-button>
            </g-tooltip>
        </div>
    </div>
</template>

mini尺寸 ​

适用于小场景或数字气泡样式。默认是large

<template>
    <div style="display: flex; gap: 10px;">
        <g-tooltip content="Hello Large Tooltip" size="large">
            <g-button>Hover Me</g-button>
        </g-tooltip>
        <g-tooltip content="Hello  Mini Tooltip" size="mini">
            <g-button>Click Me</g-button>
        </g-tooltip>
    </div>
</template>

触发方式 ​

可以通过Hover、Click、Contextmenu 三种触发方式

<template>
    <div style="display: flex; gap: 10px;">
        <g-tooltip content="Hello Tooltip" trigger="hover">
            <g-button>Hover Me</g-button>
        </g-tooltip>
        <g-tooltip content="Hello Tooltip" trigger="click">
            <g-button>Click Me</g-button>
        </g-tooltip>
        <g-tooltip content="Hello Tooltip" trigger="contextmenu">
            <g-button>Right Click Me</g-button>
        </g-tooltip>
    </div>
</template>

禁用状态 ​

禁用状态，鼠标悬停、单击、右击均无法触发文字提示。disabled 属性。

<template>
    <div style="display: flex; gap: 10px;">
        <g-tooltip content="Hello Tooltip" trigger="hover" disabled>
            <g-button>Hover Me</g-button>
        </g-tooltip>
        <g-tooltip content="Hello Tooltip" trigger="click" disabled>
            <g-button>Click Me</g-button>
        </g-tooltip>
        <g-tooltip content="Hello Tooltip" trigger="contextmenu" disabled>
            <g-button>Right Click Me</g-button>
        </g-tooltip>
    </div>
</template>

自定义控制 ​

可以通过 manual 属性手动控制。

<template>
    <g-tooltip ref="tooltip" manual content="Hello Tooltip">
        <g-button @click="toggleTooltip">Click Me</g-button>
    </g-tooltip>
</template>

<script setup>
import { ref } from "vue";

const tooltip = ref();

function toggleTooltip() {
    if (tooltip.value) {
        tooltip.value.show();
        setTimeout(() => tooltip.value.hide(), 2000);
    }
}
</script>

更多内容的文字提示 ​

展示多行文本或者设置内容的格式 通过 content 插槽设置内容

<template>
    <g-tooltip placement="top-end">
        <template #content> multiple lines<br />second line </template>
        <g-button>Top center</g-button>
    </g-tooltip>
</template>
